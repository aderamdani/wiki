# JSON Formatter

Utility untuk format dan validate JSON dengan mudah.

## Fitur

- Format JSON dengan indentation otomatis
- Validate JSON syntax
- Minify JSON
- Convert ke format lain

## Contoh

Input:
```json
{"name":"Ade","age":30,"city":"Jakarta"}
```

Output (formatted):
```json
{
  "name": "Ade",
  "age": 30,
  "city": "Jakarta"
}
```
