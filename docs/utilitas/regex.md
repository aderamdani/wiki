# Regex Tester

Utility untuk testing dan validasi regular expression.

## Penggunaan

Gunakan tools ini untuk:
- Test pattern regex
- Validate input
- Find dan replace patterns

## Contoh Pattern Umum

### Email Validation
```regex
^[^\s@]+@[^\s@]+\.[^\s@]+$
```

### Phone Number
```regex
^\+?[\d\s\-()]{7,}$
```

### URL
```regex
^https?:\/\/(www\.)?[-a-zA-Z0-9@:%._\+~#=]{1,256}\.[a-zA-Z0-9()]{1,6}\b([-a-zA-Z0-9()@:%_\+.~#?&//=]*)$
```
